# octoshell
A minishell wrtitten in C99, for educationnal purpose.

## Builtin commands
For the moment:
- `exit` : To leave the shell
- `cd` : Navigate through directories

## Builtin features
For the moment:
- `|` pipe function to pass the the STDOUT of a process to the STDIN of another
